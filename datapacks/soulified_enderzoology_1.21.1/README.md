# 🌑 SOULIFIED ENDERZOOLOGY

Dark Souls difficulty addon for **EnderZoology** mod. 8 unique mobs with elite-tier buffs and 2 mini-bosses.

---

## ⚠️ REQUIREMENTS

**Both required:**
1. **EnderZoology Mod** (Forge/NeoForge 1.21.1)
2. **Soulified Vanilla** (base datapack - required)

---

## 👹 WHAT IT DOES

**8 EnderZoology Mobs Enhanced:**

### 💀 **Mini-Bosses (Tier 3)**
- **Fallen Knight** - Strength III, Resistance II, Speed I (armored undead warrior)
- **Wither Witch** - Strength III, Resistance II, Speed I (powerful spellcaster)

### ⚡ **Elite Mobs (Tier 2-3)**
- **Concussion Creeper** - Permanently charged, Resistance II (knockback specialist)
- **Dire Wolf** - Strength II, Speed II, Resistance I (pack hunter)
- **Enderminy** - Mini enderman with Strength III, Speed I (teleporting menace)
- **Infested Zombie** - Strength II, Resistance I (spawns silverfish on hit)
- **Wither Cat** - Strength II, Speed II (applies wither effect)
- **Fallen Mount** - Skeleton horse with Strength II, Speed I

All mobs enhanced with appropriate tier buffs for mid-to-late game challenge.

---

## 📥 INSTALLATION

1. Install EnderZoology mod in mods folder
2. Install Soulified Vanilla in datapacks folder
3. Place `soulified_enderzoology` in datapacks folder
4. Run `/reload` in-game

---

## 🐛 TROUBLESHOOTING

**Entities not enhanced?** Verify EnderZoology mod is installed
**"Unknown entity type" errors?** Mod is missing - install EnderZoology
**Works on servers?** Yes, install mod + both datapacks
