# Credits & License

This is an **unofficial** addon built in the style of the "Soulified" series, applying
souls-like buffs to **Fryc Mob Variants** mobs. There is no official Soulified release
for Fryc Mob Variants; this derives from TheMrCheeky's Soulified format/conventions
(same `soulified` namespace and buff structure).
